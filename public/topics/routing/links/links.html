Manual Path Changes

When you type into the browser address bar, you are changing the path. If the new path matches an entry in the routing configuration, then the router will render the new template within the ng-view element.

Try it. Type "/binding/Notes" into the address bar (after the #).

Links

Clicking on an anchor tag also changes the path:

  • #directives/biz-card
  • #filters/Collection
  • #routing/View
  • #routing/Dynamic

Notice that the URIs in the anchor tags begin with a '#'. ngRoute supports two types of routing: hashbang and HTML5 modes: https://docs.angularjs.org/guide/$location
