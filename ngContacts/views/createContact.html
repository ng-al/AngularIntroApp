Contacts Application

Create Contact




Instructions

  1. If you have not already created the "contactService", do it per the instructions on the Introduction page.
  2. Create a "createContactCtrl" controller.
    • Use the "controller as" syntax, do not use $scope.
    • The controller will use the contactService to create a new contact.
  3. "Wire up" the template to the controller.
  4. Pressing Cancel should return you to the contact list page. (Without any data changes.)
  5. Pressing OK will return you to the contact list page where the newly added item will be in the list.

Extra Credit

Add validation to the page. The service should not be called until all of the required properties are fulfilled, and the email is properly formatted.
