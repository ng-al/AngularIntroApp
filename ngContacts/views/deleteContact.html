Contacts Application

Delete Contact

Do you really want to delete a contact?

Instructions

  1. If you have not already created the "contactService", do it per the instructions on the Introduction page.
  2. Create a "deleteContactCtrl" controller.
    • Use the "controller as" syntax, do not use $scope.
    • The controller will use the contactService to create a new contact.
  3. "Improve" the template so that it shows what contact is about to be deleted.
  4. Pressing NO should return you to the contact list page. The item will still be in the list.
  5. Pressing YES will return you to the contact list page. The deleted item will be gone.
